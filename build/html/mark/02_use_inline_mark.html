2. 内联标记¶

2.1. 标准的reST 内联标记¶

  • 星号: text 是强调 (斜体),
  • 双星号: text 重点强调 (加粗),
  • 反引号: text 代码样式.
使用限制
  1. 不能相互嵌套
  2. 内容前后不能有空白: 这样写``* text*`` 是错误的

2.2. ref¶

这个ref就是引用本项目的其他文档的连接的，需要在一个文档的顶部设置

    .. _topics-youindex_name:

在另一个文档使用 :ref:`topics-youindex_name`。
